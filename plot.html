Gatherplots data

This is a basic attempt to re-create the plots from the raw Gatherplots paper data. Iâ€™m using mixed-effects linear models instead of bootstrapping, but the overall pattern of results should be similar (and indeed they are).

Load the data

Load in the data and re-create the Visualization and Task columns:

df = read.csv("gatherplots-FINAL.csv") |>
  mutate(
    Visualization = factor(
      case_when(
        Abs & Rel ~ "both",
        as.logical(Abs) ~ "abs",
        as.logical(Rel) ~ "norm",
        .default = "jitter"
      ), 
      levels = c("both", "norm", "abs", "jitter")
    ),
    Task = factor(paste0("T", Task), levels = paste0("T", 5:1))
  )

df

Models

Since weâ€™re only interested in marginal means, weâ€™ll just do a quick-and-dirty mixed effects linear model with a Task x Visualization interaction and a random intercept by participant (I donâ€™t see multiple observations per participant per cell, so no random slopes here):

models = lapply(quos_auto_name(exprs(Correct, Time, Confidence)), function(outcome) {
  inject(lmer(!!outcome ~ Task * Visualization + (1 |ID), data = df))
})

Effects conditional on vis

First, the plots of the mean of each outcome conditional on each vis type. We can ignore the warnings emmeans gives about interactions here.

vis_plot = function(outcome_name) {
  models[[outcome_name]] |>
    emmeans( ~ Visualization) |> 
    as.data.frame() |> 
    ggplot(aes(
      x = emmean, 
      xmin = lower.CL, 
      xmax = upper.CL, 
      y = Visualization, 
      color = Visualization
    )) + 
    geom_pointrange() +
    xlab(outcome_name) +
    vis_color_scale +
    guides(color = "none")
}
vis_plot("Correct") +
  scale_x_continuous(expand = expansion(0, 0), breaks = seq(0, 1, length.out = 6)) +
  coord_cartesian(xlim = c(0, 1))
NOTE: Results may be misleading due to involvement in interactions

vis_plot("Time") +

  coord_cartesian(xlim = c(0, 70))

Effects conditional on vis x task

Now, plots of means conditional on vis and task.

vis_task_plot = function(outcome_name) {
  models[[outcome_name]] |>
    emmeans( ~ Visualization * Task) |> 
    as.data.frame() |> 
    ggplot(aes(
      x = emmean, 
      xmin = lower.CL, 
      xmax = upper.CL, 
      y = interaction(Visualization, Task, sep = " - "), 
      color = Visualization
    )) + 
    geom_pointrange() +
    xlab(outcome_name) +
    vis_color_scale +
    guides(color = "none") +
    ylab("Task | Visualization")
}
vis_task_plot("Correct") +
  scale_x_continuous(expand = expansion(0, 0), breaks = seq(0, 1, length.out = 6)) +
  coord_cartesian(xlim = c(0, 1))

vis_task_plot("Time") +
  scale_x_continuous(expand = expansion(0, 0), breaks = seq(0, 70, length.out = 8)) +
  coord_cartesian(xlim = c(0, 70))

vis_task_plot("Confidence") +
  scale_x_continuous(expand = expansion(0, 0), breaks = seq(0, 7, length.out = 8)) +
  coord_cartesian(xlim = c(0, 7))
